#Full Stack Playtime App

This project is part of my Full Stack Development coursework and demonstrates the completion of Exercise 1 of the Playtime application.

The application follows an MVC architecture using Node.js, Hapi, and Handlebars, with server-side rendering and in-memory data stores.

##Features (Exercise 1)

- User authentication (sign up, log in, log out)
- Dashboard view for logged-in users
- About page with dynamic navigation highlighting
- Playlists management (add and list playlists)
- Session-based authentication
- Handlebars layouts and partials
- Clean separation of routes, controllers, models, and views


##Tech Stack

- Node.js
- Hapi
- Handlebars (Vision)
